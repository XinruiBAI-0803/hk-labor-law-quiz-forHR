# 🇭🇰 香港劳工法自考系统 (HK Labor Law Quiz)

这是一个基于 Web 的交互式刷题工具，旨在通过**主动回忆 (Active Recall)** 的方式，辅助记忆《香港雇佣条例》中的复杂法条与逻辑。

## 💡 项目背景
作为一名雇佣条例学习者，我发现单纯阅读法条难以形成深度记忆。因此，我开发了这个轻量级的刷题工具，将枯燥的法律文本转化为结构化的数据，通过“情景判断”和“填空”的方式进行强化训练。

## 🛠️ 技术实现
本项目采用原生前端技术栈构建，无服务器依赖，轻量且高效：
- **核心架构**: HTML5 + CSS3 + Vanilla JavaScript
- **数据结构**: 使用 JSON 格式 (`data.js`) 实现题目逻辑与界面渲染的分离，便于随时扩展题库。
- **部署**: GitHub Pages

## ✨ 功能特点
1. **多题型支持**: 支持单选题、情景判断题、关键数字填空题。
2. **即时反馈**: 提交答案后立即显示正误，并展开详细的“法条解析”。
3. **响应式设计**: 适配手机与电脑端，随时随地利用碎片时间刷题。

## 📂 如何使用
1. 下载本项目代码。
2. 打开 `data.js` 文件，按照现有格式添加新的法律题目。
3. 打开 `index.html` 即可开始测试。
