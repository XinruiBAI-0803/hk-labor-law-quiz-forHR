**HK Employment Ordinance Quiz** is a web-based interactive tool designed to assist in memorizing complex clauses and logic of the *Hong Kong Employment Ordinance* through **Active Recall**.


###  Background
As a learner of the Employment Ordinance, I found that simply reading legal texts makes it difficult to form deep memories. Therefore, I developed this lightweight tool to convert dry legal texts into structured data, facilitating intensive training through "Scenario Judgment" and "Fill-in-the-blanks".

### Tech Stack
Built with a native frontend stack, zero server dependencies, lightweight and efficient:
- **Core Architecture**: HTML5 + CSS3 + Vanilla JavaScript
- **Data Structure**: Uses JSON (`data.js`) to separate quiz logic from view rendering, allowing for easy expansion of the question bank.
- **Deployment**: GitHub Pages

### Key Features
1. **Multi-Type Support**: Supports Multiple Choice, Scenario Judgment, and Key Figure Fill-in-the-blanks.
2. **Instant Feedback**: Displays correctness immediately upon submission and expands detailed "Legal Explanations".
3. **Responsive Design**: Optimized for both mobile and desktop, allowing for study sessions anytime, anywhere.

### How to Use
1. Clone or download this repository.
2. Open `data.js` and add new legal questions following the existing format.
3. Open `index.html` to start the quiz.
